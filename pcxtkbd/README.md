# pcxtkbd for AKAB interface
PS/2 -> PC/XT Arduino based Keyboard adapter.

Stable version.


Please add https://github.com/techpaul/PS2KeyAdvanced Arduino library to compile.

PCXT DIN 5 keyboard plug pinout
-------------------------------

 |                   |
/  o   o   o   o   o  \
|                     |
-----------------------
  CLK DAT NC  GND VCC
  -------------------
